# This file will explain the implementation of Data Center
1) Run bat file of the project.
2) Enter the number of VM / Servers you want to create.
3) Enter the size of each Server you want to specify.
4) Enter the RAM of each Server you want to specify.
5) Enter the Instruction processing speed of each Server you want to specify.
6) Enter the Bandwidth of each Server you want to specify.
7) Enter the no of cloudlets / tasks you want to give to the servers to execute.
8) The output for each tasks / cloudlets will be displayed after the simulation.
